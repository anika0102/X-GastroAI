Developed and deployed an end-to-end deep learning application for gastric cancer screening using histopathology images. Fine-tuned a ResNet-50 Convolutional Neural Network (CNN) with transfer learning for binary image classification (Normal vs. Cancerous Tissue). Integrated Grad-CAM (Gradient-weighted Class Activation Mapping) to provide explainable AI (XAI) visualizations for model interpretability and prediction on malignant tissue regions. Built an interactive Streamlit web application supporting image uploads, sample images, confidence scores, real-time inference, and heatmap-based decision visualization.
Bridging interpretability and medical reliability
